Dado Toolpath Creator Version 0.0

This gadget creates dado (housing joint) toolpaths from pre-selected rectangular vectors.

It is often easier to draw a part using a rectangle to layout where a dado goes but, when it comes to cutting, it is more efficient to use the correct diameter bit and do a profile cut on a centre line.

This gadget automates the process of creating those centre lines and toolpaths for the selected tool size.

To use the gadget select one or more rectangular vectors. The gadget assumes that the dado will run in the direction of the longest side of the rectangle.

The Allowance is the amount you want the centre line to over or under shoot the rectangle. Typically you would want the line to be longer than the rectangle by the tool diameter if you were cutting a full width dado. If the allowance is left blank then it will default to the selected tool diameter.

If you want to use an existing toolpath template for the dadoes then uncheck the Create Toolpath check box and only the vectors will be created on the specified layer.
