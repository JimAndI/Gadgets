Spiral Layout Version 0.0
This gadget creates vectors which can be used to machine spirals along a column (rope or barley twist). Once the vectors have been created, use the 2D Profile Toolpath with the 'Machine Vectors .. On' form within the program to create the actual toolpaths at your required depths.
